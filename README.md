# Gabriella & Daniel – Bröllopssida

En enkel, statisk hemsida för bröllopet den 20 juni 2026 i Ryssby.

## Så här öppnar du sidan

- **Snabbast**: dubbelklicka på `index.html` och öppna den i din webbläsare.
- **Med enkel dev-server** (kräver Node och npm):
  1. Öppna en terminal i den här mappen.
  2. Kör:

     ```bash
     npm install serve --save-dev
     npm run start
     ```

  3. Öppna den adress som skrivs ut i terminalen (oftast `http://localhost:3000`).

## Ändra innehåll

- **Texter**: Redigera sektionerna direkt i `index.html` (svenska/kroatiska).
- **Bilder**: Byt ut bildfilerna i mappen `assets` eller uppdatera sökvägarna i `index.html`.
- **Färger & typsnitt**: Justeras i `styles.css`.

Kom ihåg att spara filerna och ladda om sidan i webbläsaren för att se ändringarna.
